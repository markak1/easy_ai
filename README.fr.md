# EASY-AI

**Plateforme d'Accès Unifié aux Modèles IA**

</div>

## À Propos

EASY-AI est une plateforme de relais et de gestion de modèles IA prenant en charge l'accès unifié, la distribution et la facturation des principaux LLM.

## Fonctionnalités

- **Support Multi-Modèles** — OpenAI, Claude, Gemini, DeepSeek, Qwen, etc.
- **API Compatible OpenAI** — API standardisée
- **Gestion des Canaux** — Basculement automatique, nouvelle tentative automatique
- **Système de Permissions** — Groupement de tokens, restrictions de modèles
- **Facturation & Recharge** — Gestion flexible des quotas

## Déploiement

```bash
git clone https://github.com/markak1/easy_ai.git
cd easy_ai
docker-compose up -d
```

Visitez `http://localhost:3000` après le déploiement.
